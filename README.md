Este documento corresponde a una calculadora especialmente diseñada para calcular el Índice de Masa Corporal (IMC) y ofrecer recomendaciones nutricionales personalizadas. 
El repositorio asociado incluye dos archivos clave: el primero es index.html, que contiene la estructura completa de la calculadora. 
Esta calculadora ha sido desarrollada utilizando CSS para lograr un diseño estético atractivo y funcional, y JavaScript para realizar el cálculo dinámico del IMC al ingresar los datos proporcionados por el usuario directamente en la interfaz.
El segundo archivo es el logo oficial del COBAEM en formato .ico, que se utiliza como favicon en la página web, contribuyendo a la identidad visual y profesionalismo del sitio.
