Este repositorio contiene una calculadora diseñada para calcular el Índice de Masa Corporal (IMC) y ofrecer recomendaciones nutricionales personalizadas. Incluye dos archivos clave: el primero, index.html, que alberga la estructura completa de la calculadora. Esta ha sido desarrollada con CSS para un diseño atractivo y funcional, y JavaScript para calcular dinámicamente el IMC según los datos ingresados por el usuario en la interfaz. El segundo archivo es el logo oficial del COBAEM en formato .ico, utilizado como favicon en la página web, lo que refuerza la identidad visual y el profesionalismo del sitio.
